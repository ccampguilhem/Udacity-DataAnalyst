Help

Hover an airport to display tooltip, then click to filter routes from/to that airport. Click again on that airport or anywhere in then map to unselect airport.

Use inputs to configure min and max route lengths and minimum number of flights by route then click on Apply button to update visualization. You can also press ENTER key instead.

Click on an aircraft name in bar chart to filter flights of that aircraft. You can select multiple aircrafts. Click again on the selected aircrafts, which appear in a different color, to unselect them.

Use Reset button to restore default state of visualization, or Replay button to see again the animated part.
